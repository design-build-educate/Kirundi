LESSON 7: Class 3 in- in-

Vocabulary

  • inka - cow
  • Imāna - God
  • imbga - dog
  • impūzu - clothes
  • indimiro - field
  • inzu - house, room
  • intebe - chair, stool

13. Class 3. The prefix for Class 3 is in- for both singular and plural. The context and agreements will usually indicate whether the word is singular or plural. Later we will note some variations of this prefix.

Possessive adjective prefix:

sing.: ya plural: za
e.g. inka yange e.g. inka zanje
(my cow) (my cows)

14. Consonant change. (Memorize this rule: it is used in many words.)

  • N before all labials (B,M,V,F,P) becomes M - e.g. imbga (in-bga)
  • N before H becomes MP - e.g. impuzu (in-huzu)
  • N before R becomes ND - e.g. indimiro (in-rimiro)
  • N before another n drops out.
  • N before a votiel root becomes NZ - e.g. inzara (in-ara)

Note: In the word Imana (in-mana) the n drops out before m, thus making Imana.
